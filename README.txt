## CSV File Parsing Maven Application

# Pre-requisites for running this application:

- Java 8
- Eclipse or any other Java based IDE
- Maven
- CSV file (interview_data_final.csv) to be placed in the project's root directory

# About the application:

- This application is developed using Java 8. It is a console application.
- The main class had the entire functionality to parse the csv, filter the data by age, store it in a data structure (ArrayList) and display it grouped by State.
- Display functionality is achieved using two ways: Using System.out object and by creating json object
